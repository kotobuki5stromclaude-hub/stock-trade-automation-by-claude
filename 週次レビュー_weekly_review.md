# 週次レビュー(振り返り → 改善ループ)

毎週金曜にこの下へ追記(新しい週が上)。目的: 実績から損益を出し、プラス/マイナス要因を分析し、
次週の改善アクションを積み上げて運用成績をプラスに寄せていく。

## 集計の定義
- 実現損益: holdings.csv の SELL 行から算出（BUY 加重平均との差額×株数の累計）。通貨別（JPY / USD）に分けて集計。
- 含み損益: リアルタイム価格非対応のため表示なし（将来対応予定）。
- 勝率: 実現損益 > 0 の確定銘柄数 / 総確定銘柄数。
- 判断の検証: trade_decisions.csv の根拠と実際の結果を突き合わせる。

## 各週セクションの型
- 損益サマリー(実現/含み, 勝率)
- ◎ プラス要因(うまくいった判断・銘柄・テーマ)
- × マイナス要因(損切り/高値掴み/見送りミス/入りどころ 等)
- → 学び・次週の改善アクション(1〜3個、具体的に)
